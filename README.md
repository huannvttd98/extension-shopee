# ProductMap

Crawl sản phẩm Shopee bằng Chrome Extension (MV3) + Backend FastAPI + MySQL.

## Mục tiêu

1. Crawl data Shopee
2. ~1 triệu sản phẩm theo chủ đề
3. API tìm kiếm sản phẩm sau khi crawl *(phase sau)*
